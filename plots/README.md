# Training Plots

This directory contains visualization plots generated by running `train.py`.

## Files

- `training_loss.png` - Training and validation loss curves over epochs
- `training_accuracy.png` - Training and validation accuracy curves over epochs  
- `training_curves.png` - Combined plot showing both loss and accuracy side-by-side

These plots help visualize the baseline model's training behavior.

To regenerate these plots, simply run:

```bash
uv run train.py
```
